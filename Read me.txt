This repository contains an independent, self-initiated data analysis project examining sectoral inflation 
Sectoral Inflation Dynamics in Europe (2024–2025)

dynamics in selected European countries between November 2024 and October 2025.

Objective

To analyse short-run price dynamics, cumulative inflation, and cross-country heterogeneity across major HICP consumption categories.

Data

Monthly Harmonised Index of Consumer Prices (HICP, 2015=100) from Eurostat for selected European countries.

Methods
 • Descriptive statistics
 • Sectoral time-series analysis
 • Cross-country comparison of cumulative inflation
 • Python (Pandas, Matplotlib)

Structure
 • data/: Raw HICP data by consumption category
 • notebooks/: Jupyter notebooks for data processing and analysis
 • figures/: Final visualisations
 • paper/: 2-page PDF report

Note

This project was conducted independently and is not part of any university coursework.
